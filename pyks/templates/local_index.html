Welcome to PyKS - Proxy your Key Servers. This is a locals proxy to provide key services to your existing OpenPGP clients like query for public key certificates of your known contacts, check if their public key certificate is still valid or already revoced etc. The OpenPGP HTTP Keyserver Protocol (HKP) allows OpenPGP clients to interoperate with different keyserver implementations. You can learn more about the HKP in the Test Cases section.

You can look at the locally stored data that can be queried by your OpenPGP client. Later you can also look at the remote stored data.
